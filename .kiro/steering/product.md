# RBAC Admin Pro

企业级全栈后台管理系统，基于 RBAC (Role-Based Access Control) 权限模型。

## 核心功能

### 系统管理
- 用户管理：CRUD、角色/部门分配、状态控制、密码重置
- 角色管理：权限配置、菜单绑定
- 部门管理：树形组织架构
- 菜单管理：动态菜单、按钮级权限
- 岗位/字典/参数/公告管理

### 系统监控
- 操作日志、登录日志
- 在线用户管理（支持强制下线）
- 服务器状态、缓存监控
- 定时任务管理

## 语言规范
- UI 界面：简体中文
- 代码注释：中文
- API 响应消息：中文
- 文档：中文
- Git 提交信息：中文（格式：`类型: 描述`，如 `feat: 新增用户管理功能`）

### Git 提交类型
- `feat`: 新功能
- `fix`: 修复 Bug
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 重构
- `perf`: 性能优化
- `test`: 测试相关
- `chore`: 构建/工具变动
