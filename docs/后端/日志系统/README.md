## 📊 系统概览

### 已集成模块
- ✅ 11个 Service 模块
- ✅ 3个系统组件
- ✅ 覆盖所有核心业务

### 日志文件
- `application-*.log` - 所有日志
- `error-*.log` - 错误日志
- `http-*.log` - HTTP请求
- `exceptions-*.log` - 未捕获异常
- `rejections-*.log` - Promise拒绝

### 日志级别
- **error** - 系统错误
- **warn** - 警告、敏感操作
- **log** - 重要业务操作
- **http** - HTTP请求
- **debug** - 调试信息

## 💡 快速命令

```bash
# 查看实时日志
tail -f logs/application-*.log

# 查看错误日志
tail -f logs/error-*.log

# 搜索日志
grep "关键词" logs/application-*.log
```
